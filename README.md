A modern, responsive frontend dashboard designed to streamline and automate attendance tracking. Built with a focus on clean UI/UX, this system provides real-time insights for administrators and users.

🚀 Features
Live Dashboard: Real-time overview of attendance percentages and daily statistics.

User Management: Easy-to-navigate interface for managing student or employee profiles.

Detailed Analytics: Visual representation of attendance trends using interactive charts.

Search & Filter: Quickly find specific records by date, department, or status.

Responsive Design: Fully optimized for desktops, tablets, and mobile devices.

🛠️ Tech Stack
Framework: Next.js (React)

Styling: Tailwind CSS

Components: shadcn/ui

Icons: Lucide React
